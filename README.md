# BrnoJUG 2025 - slides and examples

This repository contains slides and examples for the [« Building efficient MCP servers efficiently, with Quarkus »](https://www.meetup.com/brno-java-meetup/events/311404698/) session.

## Slides

1. Install [Quarkus Reveal](https://github.com/quarkusio/quarkus-reveal/)
2. Run the following command to see the MCP slides: `quarkus-reveal slideas/build_mcp_servers.md -t dev`

## Examples

Run `mvn quarkus:dev` in the relevant example dir ;-).
